# Real Time Chat (Vue, node)

> front-end: Vue.js
>
> back-end: node.js

**server module**

- socket.io
- express

**front module**

- socket.io-client

### Server Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:3001
node app.js
```



### Vue Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```
