# Real Time Chat (Vue, node)

> front-end: Vue.js
>
> back-end: node.js

### Overview

뷰로 socket.io를 적용하여 채팅 시스템을 만들었다.

우선 구현을 위하여 코드를 더럽게 짜서 최적화가 필요하다.

**server module**

- socket.io
- express

**front module**

- socket.io-client

### Server Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:3001
node app.js
```



### Vue Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```
